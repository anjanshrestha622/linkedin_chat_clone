## LinkedIn Chat Clone using TalkJS

| Tasks | Commands |
| ------ | ------ |
| Start | npm start |
| Test | npm test |
| Lint | npm run lint |
| Build | npm run build |

<h4>Basic config for .env file</h4>

| Environment variables | TalkJS a/c |
| ------ | ------ |
| REACT_APP_APP_ID | App Id |
| REACT_APP_SECRET_KEY | Secret key  |
